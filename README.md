# 🐱 Cat Chaos Simulator
A high-energy, retro pixel-art arcade game built with vanilla JavaScript and the Web Audio API.

## 🎮 How to Play
- **Objective:** Collect items to score points and build your combo while avoiding falling furniture!
- **Controls:** Use `Arrow Keys` or `A/D` to move the cat.
- **Scoring:** - 🍕 Pizza: 5 pts
  - 🐟 Fish: 10 pts
  - ⭐ Star: 15 pts
- **Danger:** Don't let the bombs, chairs, or pianos bonk you! You have 9 lives.

## 🚀 Features
- **Procedural Audio:** Sound effects are synthesized in real-time using the Web Audio API (no external MP3 files needed!).
- **Custom Pixel Engine:** All sprites and fonts are rendered via HTML5 Canvas pixel-by-pixel.
- **Combo System:** Fast play is rewarded with score multipliers.

## 🛠️ Technical Details
- **Language:** JavaScript (ES6+)
- **Graphics:** HTML5 Canvas
- **Audio:** Web Audio API
- **Zero Dependencies:** No engines or libraries used.
